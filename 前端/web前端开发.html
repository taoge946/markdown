div与span

  1. div定义：div是分区或分节的意思是一个块级标记,以< div>开始以< /div>结束
    用法为< div id=""class=""style="">块包含的内容
    style可取的属性和值的含义:
    style属性 值 说明
    position static 表示静态定位，默认设置
    absolute 表示绝对定位，与位置属性配合使用
    relative 表示相对定位，图层不可层叠
    fixed 表示图层位置固定，不滚动
    inherit 规定应该从父元素继承 position 属性的值。
    border 线粗细，线型，颜色 边框，可设置风格，粗细，颜色等属性，可按照宽度，样式，颜色一齐定义，如:border:5px solid red;
    background-color rgb(),#ffffff,英文颜色名 背景颜色
    left(top) px或% 规定层左边(顶部)的距离
    width(height) px或% 规定层的宽度（高度）
    float right，left，none 允许浮动元素在左右以及不浮动。float 属性定义元素在哪个方向浮动。以往这个属性总应用于图像，使文本围绕在图像周围，不过在 CSS 中，任何元素都可以浮动。浮动元素会生成一个块级框，而不论它本身是何种元素。如果浮动非替换元素，则要指定一个明确的宽度；否则，它们会尽可能地窄。
    clear left,right,both,none 分别清除左边右边左右两边的浮动和允许左右两边有浮动
    z-index auto或数字 表示子层会按照父层的属性显示/无单位的整数或负数
    overflow scroll,visible,auto,hidden 内容溢出控制。分别表示显示滚动条，不显示滚动条，但超出部分可见，内容超出时显示滚动条，超出时隐藏内容
    display block,inline,none 表示按块元素显示，行内方式显示和隐藏
  2. 图层嵌套与层叠
    1. 嵌套:多个div可单独使用也可以互相包含，嵌套使用。一方面可以把页面分成不同的块，另一方面可以把功能相近的块组织到一个更大的块中便于整体控制。
    2. 层叠：必须要先将div的position属性设置为absolute（div{position:absolute;}），再利用z-index属性控制层叠关系
  3. span:span是行内标记，用法和div相同，可通过display转化为块标记

    1. 绝对单位：绝对单位在网页中很少使用，但在特殊场合用绝对单位是必要的，绝对单位包括：英寸(in)厘米(cm)毫米(mm)磅(pt)pica(pc)(1pc=12pt)
    2. 相对单位：显示大小不固定，受分辨率或浏览器设置等因素的影响，css中常使用的相对单位包括em，ex，px，%
      • em:表示元素的字体高度，他能根据父元素的font-szie值来确定
      • ex:em的值除以二
      • px:根据屏幕像素点来确定的，不同的分辨率下显示的样式不同。建议整个网页中只出现px或em来实现单位的统一。
      • 是一个相对单位值，一般参考父元素中相同属性的值。

  3. css table属性
    属性 说明
    border-collapse 是否把表格边框合并为单一的边框
    border-spacing 设置分隔单元格的距离
    caption-side 设置表格标题的位置
    empty-cells 设置是否显示表格中空的单元格
    table-layout 设置显示单元行和列的算法
    • border-collapse属性：默认表示表格具有双线边框，可以通过这个属性使表格变为单线框有以下的值：
      separate:默认值，边框会分开不会忽略border-spacing和empty-cells属性

      • collapse:边框合并为一个单一的框
      inheirt:从父元素那里继承
    • border-spacing属性：用于设置或检索当表格边框独立时，行和单元格的边框在横向和纵向上的间距。该属性等同于标签属性cellspacing基本用法border-spacing:length length
      只有在border-collapse的值为separate时才会生效
      如果两个length全部提供那么第一个作用于横向间距，第二个作用于纵向间距
    • caption-side属性：设置表格的标题在哪一边(bottom/left/right/top)
    • empty-cells属性:设置或检索当表格的单元格无内容时是否显示该单元格
      只有在border-collapse的值为separate时才会生效
    • table-layout属性：设置或检索表格的布局算法
      auto:默认自动布局算法，将基于各单元格的内容。表格在每一个单元格读取计算之后才会显示出来
      fixed:固定布局的算法,在这个算法中，水平布局仅仅是基于表格的宽度，表格边框的宽度，单元格间距，列的宽度与表格内容无关（表格只和设置的宽度有关）。也就是说内容可能被裁切

框架

  1. 框架概述：可以在一个网页中放置多个网页，以< frameset>为开头以< /frameset>为结尾中间用< frame src="">来放置目标网页。有frameset的网页中没有body标签，如框架1text就是一个框架案例
    代码为：< frameset rows="10%,80%,10%">
    < frame src=""name="top">
    < frameset cols="20%,*">
    < frame src=""name="middle_left">
    < frame src=""name="middle_right">
    < /frameset>
    < frame src=""name="bottom">
    < /frameset>
  2. 框架集属性：
    属性 frameborder framespacing border bordercolor rows cols
    说明 设置框架集的显示效果(可设为0或1来表示边框的显示与否) 设置框架间的间距（宽度） 设置框架集边框的宽度 设置框架集边框的颜色 设置框架集的水平分割 设置框架集的垂直分割
    分割可分为具体数值或是百分比或是用"*"号代替之间用逗号","链接，如< frameset rows="30%,20%,*">
  3. 框架的设置
    1. 框架名称属性name:可以给每个frame标签一个名称，可做超链接的target的属性值，用法：< frame src=""name="">
    2. 框架中的滚动条属性scrolling:yes/no/auto 可以根据此属性来设定是否显示属性条
    3. 框架的边距marginwidth和marginheight属性：定义框架的左侧右侧和上侧下侧边距
    4. 框架属性调整属性noresize:属性值可以时noresize也可以不赋值，表示不允许调整框架的大小
  4. 浮动框架：浏览器窗口含有独立的子窗口称为浮动框架，也称内联框架。在浏览器窗口中嵌入浮动框架可用iframe标记，该标记为成对标记，必须插入在body标记中，而不能插入frameset标记中。iframe标记的属性值如下：
    • src:设置源文件属性
    • name:设置框架名称
    • width,height,align
    • frameborder(设置框架边框)scrolling(设置框架滚动条)marginwitdh,marginheight
    基本语法：< iframe 属性名称=“”>< /iframe>
